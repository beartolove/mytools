首次設定

需要 Google OAuth Client ID 才能存取你的 YouTube 資料。
設定只需 5 分鐘，且完全免費。

設定步驟： 1. 前往 Google Cloud Console
2. 建立新專案（或使用現有專案）
3. 搜尋並啟用 YouTube Data API v3
4. 前往「憑證」→「建立憑證」→「OAuth 2用戶端ID」
5. 類型選「網頁應用程式」
6. 「已授權的 JavaScript 來源」加入此網頁的網址
7. 複製 Client ID 貼入下方
